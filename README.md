# My-Daily-Prayers-App
Daily Prayer Reminder  Daily Prayer Reminder is a simple and elegant desktop application that helps users keep track of their daily prayers. The app allows you to mark prayers as completed, view descriptions, reset progress, and even toggle between dark and light themes.
Features.

- Track 5 daily prayers: Fajr, Dhuhr, Asr, Maghrib, Isha

- Mark prayers as completed and see your progress

- View descriptions of each prayer

- Reset daily prayers

- Toggle between Dark and Light themes

 - secure login with password protection

# Windows Installer Available

# A standalone Windows installer is available, so Python is NOT required on your computer. You can install and start using the app immediately.

# How to Install

# Download the latest Windows installer from the Releases page.
 

# Double-click the .exe file to launch the installer.

# Follow the on-screen instructions to complete the installation.

# Launch Daily Prayer Reminder from your Start menu or desktop shortcut.

Enjoy keeping track of your daily prayers effortlessly!
